### 記法ルール
- 英字と日本語の間に半角スペースを設ける (記号は例外)
例: "Git とは"
- ()の前に半角スペースを設ける (ただし文章中の場合は例外とする)
例: "Summit (KDE の翻訳システム)"
例外: "りんご(トマトかも)がおいしかった"
- 英字、記号などはすべて半角
例: "build します!"
- 拡張子は code 記法を利用する
例: "`.html`"
